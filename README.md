## Overview
This project analyzes retail sales data to derive insights for demand analysis,
product prioritization (ABC analysis), and inventory planning.

## Approach
- Aggregated transactional sales data at product level
- Performed ABC analysis based on revenue contribution
- Analyzed monthly demand trends
- Applied a simple moving average for demand forecasting
- Estimated Economic Order Quantity (EOQ) using standard assumptions

## Dataset
Source: https://www.kaggle.com/datasets/sloozecareers/slooze-challenge  
Raw CSV files are stored in the `data/` directory and were not modified.

## Output
The main analysis is provided in `analysis_pipeline.ipynb`.
